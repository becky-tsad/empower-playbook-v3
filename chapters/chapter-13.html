The Empower Toolkit

Everything you need to move from reading this playbook to implementing the model, organized by implementation phase.

Program Managers 5 min read

Everything you need to move from reading this playbook to implementing the model. These tools are organized by implementation phase.

13.1

Assessment Tools (Before You Start)

Use these before committing to an Empower implementation. They determine whether your community is ready — and where you need to build capacity first.

Tool Purpose When to Use
Community Readiness Assessment Evaluates existing service infrastructure, employer landscape, political climate, and organizational capacity Before committing to implementation
Service Mapping Template Catalogs every agency, program, and funding stream touching your target population — identifies duplications and gaps Phase 1, first month
Employer Landscape Analysis Maps employers by sector, pay scale, benefits, growth potential, and openness to certification Phase 1, before launching employer program
Benefits Cliff Baseline Calculator Assesses the severity of benefits cliffs in your state's specific policy environment Phase 1, informs navigator training priorities
13.2

Planning Tools (Phase 1)

Tool Purpose When to Use
Shared Vision Workshop Guide Facilitates the alignment session with all partner organizations — produces the signed shared vision document Phase 1, first month
Phase 1 Master Checklist Complete task list for foundation-building phase with success markers Phase 1, months 1–6
Partner Agreement Templates Coordination agreements specifying roles, data sharing, meeting cadences, and accountability Phase 1, before enrollment begins
Job Description Templates Templated descriptions for all four core roles + supporting roles, calibrated to Empower methodology requirements Phase 1, hiring period
Budget Framework Scalable budget template at three implementation sizes: small pilot (50 families), medium (125), full (250+) Phase 1, funding planning
13.3

Implementation Tools (Phase 2–3)

Tool Purpose When to Use
Navigator Training Curriculum Complete training sequence: methodology, benefits cliff tools, ALICE orientation, coordination protocols Phase 1 (training) through ongoing
Benefits Cliff Calculator Access State-specific cliff modeling tool — or connection to the Beacon technology platform From first enrollment
Employer Certification Checklist Step-by-step requirements and verification for certified employer status Phase 1–2, employer onboarding
Poverty IQ Training Materials Employer-facing training on ALICE realities, benefits cliff effects, and retention strategies Phase 1–2, employer certification
Intake Assessment Template Comprehensive family assessment covering income, benefits, employment, childcare, transportation, health coverage, housing Phase 2, first enrollment
Transition Planning Worksheet Per-family benefits cliff transition plan — sequences income increases against benefit reductions Phase 2–3, active navigation
13.4

Data Tools

Tool Purpose When to Use
Dashboard Setup Guide Technical requirements and configuration for unified tracking system Phase 1
Data Collection Templates Standardized intake forms, progress tracking, and outcome documentation Phase 2 onward
Outcomes Reporting Framework Defines what to measure, how to measure it, and how to report it for funders and leadership Phase 2 onward
ROI Calculation Methodology Step-by-step guide to calculating return on investment using the Urban Institute-aligned approach Phase 3, outcomes documentation
13.5

Sustainability Tools

Tool Purpose When to Use
Grant Prospect Research Template Framework for evaluating potential funders: fit, capacity, timeline, likelihood From Phase 1
Funder Pitch Deck Template Customizable presentation framework for foundation meetings Active fundraising
ROI One-Pager Template Single-page summary of outcomes for quick funder distribution Active fundraising
Success Story Collection Guide Ethical storytelling protocols for gathering and using participant narratives Phase 2 onward
Funder Segmentation Matrix Template for matching your data points to specific funder priorities (see Chapter 10) Active fundraising
