Green fails most contrast tests — this is why we use it only for large decorative elements (stat numbers on dark backgrounds), never for small text or interactive elements.

When to Use Each Color

Do

  • Use Navy for headings, key takeaway borders, dark backgrounds
  • Use Blue for links, active states, section numbers, callout borders
  • Use Charcoal for body text, secondary labels, neutral borders
  • Use Green only for stat numbers in data boxes and small progress indicators
  • Use blue/navy tints (50-100) for subtle backgrounds

Don't

  • Don't use green for text, buttons, links, or navigation
  • Don't use green backgrounds (white text on green fails WCAG)
  • Don't use gradients on chapter headers
  • Don't use more than 2 colors in any single component
  • Don't tint body content backgrounds — keep them white or gray-50
